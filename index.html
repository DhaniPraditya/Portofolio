About Me.

I’m a UI/UX Designer with an Information Technology background from Yogyakarta State University.

During my internship at PT Nusantara Regas, I contributed to designing an internal company website that improved usability and visual consistency. I’m passionate about creating user-centered designs that are both functional and visually cohesive. One of my key projects is the redesign of the UNY Registration System, where I emphasized consistency and responsiveness by leveraging Figma’s component and auto layout features. I value collaboration, critical thinking, and solving real problems through design.

Selected Works.

Tugboat Daily Report

Web Design

This project aimed to digitize the tugboat daily reporting process, which was previously managed using Excel, into a centralized internal web platform for easier monitoring and data checking.

I was responsible for conducting user research, creating wireframes, and developing interactive prototypes in Figma. Throughout the process, I collaborated closely with the front-end and back-end developers to ensure the design was feasible for implementation. By the end of my internship, the project had entered the development phase based on my finalized design. Project Link.

Social Commerce Analytics

Web Design

Designed a definitive analytics intelligence platform for social-commerce, enabling brands, agencies, and creators to measure, optimize, and scale performance across TikTok Shop, Shopee, Tokopedia, and emerging platforms.

As UI/UX Designer, I built a design system and produced 22+ wireframes evolving into interactive Figma prototypes. The solution tackles core pain points: creators can’t spot revenue-driving trends, marketers struggle to identify high-ROI creators, and engagement alone can’t compare campaign effectiveness. The platform aligns insights to sales outcomes, prioritizing valuable content over “just viral.” Project Link.

Yogyakarta State University Registration System

Web Redesign

This project focused on improving the usability and user experience of Yogtakarta State University's registration website, which previously had confusing navigation and inconsistent visuals.

Through the Design Thinking process—starting from user surveys (SUS & UEQ), problem definition, ideation, and wireframing to prototyping in Figma—I created a responsive and consistent interface using components and auto layout. Usability testing results showed significant improvement, with both usability and user experience rated “Good” to “Excellent.” Project Link.
